## Features

<CardGrid stagger>
  <Card title="Chat Broadcasts" icon="chat">
    Create interactive chat messages where players can click to claim random amounts of money. Perfect for special events!
  </Card>

  <Card title="Physical Envelopes" icon="approve-check">
    Generate custom physical items (using ItemsAdder/Oraxen/Nexo) that players can trade, gift, or save for later.
  </Card>

  <Card title="Direct Transfers" icon="forward-slash">
    Send money directly to other players with a simple command. Fast and secure.
  </Card>

  <Card title="Folia Compatible" icon="rocket">
    Built with FoliaLib for full compatibility with both Paper and Folia servers. All operations are async.
  </Card>
</CardGrid>
